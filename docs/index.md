# RAPID
> An open-source RF AI-Pipeline for Integrated Circuit Design

<div style="display:flex; align-items:center; gap:1.5rem; flex-wrap:wrap;">
	<img src="logo.png" alt="RAPID logo" width="300"/>
	<p style="margin:0; flex:1; min-width:240px;">The RAPID project is dedicated to developing a toolchain for the design and simulation of RFICs using artificial intelligence techniques and open-source software. It provides a modular framework for generating geometries, running electromagnetic simulations, and training machine learning models to optimize RFIC designs. It provides a GUI and several pre-trained models for easy usage.</p>
