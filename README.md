# ORCA
> **O**pen **R**F Integrated **C**ircuit **A**utomation

The ORCA project is dedicated to developing a toolchain for the design and simulation of RFICs using artificial intelligence techniques and open-source software. It provides a modular framework for generating geometries, running electromagnetic simulations, and training machine learning models to optimize RFIC designs. A GUI and several pre-trained models are included for easy usage. The Documentation can be found [here](https://davidl-11.github.io/ORCA/)

For easy deployment, we also supply an OpenStack image and a CLI controller that allows you to launch simulations and training jobs on a remote server
in our [other repository](https://github.com/DavidL-11/ORCA-OpenStack).
