# ORCA
> **O**pen **R**F Integrated **C**ircuit **A**utomation

The ORCA project is dedicated to developing a toolchain for the design and simulation of RFICs using artificial intelligence techniques and open-source software. It provides a modular framework for generating geometries, running electromagnetic simulations, and training machine learning models to optimize RFIC designs. It provides a GUI and several pre-trained models for easy usage.

Documentation can be found [here](https://davidl-11.github.io/ORCA/)
